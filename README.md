## How to Compile:

# Use GCC to compile any question:

gcc questionN.c -o questionN

Example:

gcc question1.c -o question1

# To Run (after compiling):

./question1

Notes

Each question has its own file and compiles independently.

Enter multiple inputs with spaces, not commas.

Question 10 requires 9 numbers for Matrix A and 9 numbers for Matrix B.
